# Shopping_Cart
This is a project for Java Lesson.

## 环境配置

1. 安装JDK12或17，并安装idea作为IDE
2. 命令行终端`git clone https://github.com/Mick4994/Shopping_Cart`
3. 修改项目文件编码为`GBK`，并设置项目SDK
4. 设置启动配置：
* 服务端：ProductDataServer类，添加VM参数改编码为GBK
* 客户端：Ascentsys类
